# SpotiFlávio

Player de som utilizando o framework AV Foundation, Swift 4, Xcode 10. Consumindo músicas localmente no app, no 
formato .mp3. Mostrando o álbum e nome do artitsta e nome da música. No exemplo, temos 3 músicas.
